  - username
  - passwordHash
  - salt

  - Verifier

------------------------------------------------
3. ENCRYPTION & KEY EXCHANGE MODULE
------------------------------------------------
- Implement AES encryption & decryption
- Implement RSA key generation

- VISUALIZE step-by-step:
  Message → Key → Encrypted → Decrypted

4. HASHING & DIGITAL SIGNATURE MODULE
------------------------------------------------
- Implement SHA-256 hashing
- Implement hashing with salt
- Implement Digital Signature:
  - Hash data
